Project 3 - VSM Cosine Retrieval Evaluation
===========================================

Corpus     : Financial Times 1991 (FT911) - 5,368 documents
Cut-off    : top 50 ranked documents per query
Weighting  : lnc.ltc (Manning et al. Fig 6.14)
            documents -> (1 + log10 tf)
            queries   -> (1 + log10 tf) * log10(N / df)
Stopwords  : SMART stopword list (504 entries)
Stemmer    : Porter (1980)

'Relevant (corpus)' is the number of documents judged relevant in the qrels
file that are actually present in the indexed corpus (FT911 only).  Recall
is computed against this in-corpus relevant set so it reflects what is
achievable given the available document collection.
